# Cambios V15 - Precisión desde Japan vs Tunisia y Fixture ID manual

### 1. Precisión acumulada desde Japan vs Tunisia
La pestaña Precisión ahora empieza a contar desde:
- Japan vs Tunisia / Tunisia vs Japan
- fixture interno 66456974
- fecha base 2026-06-21

Los partidos anteriores ya no se cuentan en el porcentaje acumulado.

### 2. Fixture ID manual conservado
El campo "Fixture ID opcional" ya no se sobrescribe automáticamente con el calendario interno.
Por defecto conserva el ID manual indicado: 236.
Si quieres usar otro, puedes cambiarlo en la interfaz.
Si lo borras, la app usa el calendario interno.

### 3. Mejor mensaje en la pestaña Precisión
Si aún no hay partidos completados desde Japan vs Tunisia, la app muestra un aviso claro en lugar de mezclar partidos anteriores.
