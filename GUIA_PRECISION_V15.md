# Guía V15 - Precisión desde Japan vs Tunisia

## Desde dónde empieza la precisión

La medición de precisión empieza desde:

Japan vs Tunisia / Tunisia vs Japan

Esto significa que partidos anteriores como:
- United States vs Australia
- Scotland vs Morocco
- Brazil vs Haiti
- Turkey vs Paraguay

no entran en el porcentaje de precisión acumulada.

## Regla de acierto

Si el modelo dice que el resultado más probable es:
- gana Equipo A
- empate
- gana Equipo B

y eso coincide con el resultado real, cuenta como acierto.

## Fixture ID manual

El campo Fixture ID opcional conserva el valor 236 por defecto.

Si quieres usarlo para consultar una fuente externa o una API alternativa, lo puedes dejar.
Si no quieres usarlo, puedes borrarlo y la app seguirá con el calendario interno.
